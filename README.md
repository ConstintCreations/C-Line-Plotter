# C Line Plotter

This is a simple terminal-based line plotter made in C for plotting lines in the y = mx + b format. This was only tested on the Konsole terminal for the KDE desktop environment.

![C Line Plotter](/attachments/c-line-plotter.png)

## Usage

You can either run the line plotter and input numbers when prompted, or input them as arguments like so:

```
    line_plotter [slope] [y-intercept]
```
